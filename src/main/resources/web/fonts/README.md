Self-hosted UI fonts live in this directory.

Installed for the control plane:

- `IBMPlexSans-Regular.woff2`
- `IBMPlexSans-Medium.woff2`
- `IBMPlexSans-SemiBold.woff2`
- `IBMPlexMono-Regular.woff2`
- `IBMPlexMono-Medium.woff2`
- `IBMPlexMono-SemiBold.woff2`

Preferred family pair for the control plane:

- `ibm-plex-sans-400.woff2`
- `ibm-plex-sans-500.woff2`
- `ibm-plex-sans-600.woff2`
- `ibm-plex-mono-400.woff2`
- `ibm-plex-mono-500.woff2`

When these files are added, register them from `styles/app.css` with `@font-face`
and keep `font-display: swap`.
